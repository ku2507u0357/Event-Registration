# 🎓 Cloud-Based College Event Registration System

## 📌 Project Overview
This project is a web-based application that allows students to register for college events online.  
It replaces manual registration with a digital system and stores data in a cloud database in real time.

---

## 🎯 Objectives
- Create an easy-to-use event registration system  
- Store user data in the cloud  
- Make the application publicly accessible  
- Implement a simple serverless architecture  

---

## 🏗️ Architecture

User (Browser)  
↓  
Frontend (HTML, CSS)  
↓  
Backend (Flask - Python)  
↓  
Firebase Realtime Database  

---

## ☁️ Technologies Used

- **Frontend:** HTML, CSS  
- **Backend:** Python Flask  
- **Database:** Firebase Realtime Database  
- **Hosting:** Render  

---

## ⚙️ Features

- User-friendly registration form  
- Multiple event selection  
- Input validation  
- Real-time data storage in Firebase  
- Success message after submission
